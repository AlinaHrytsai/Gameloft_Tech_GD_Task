Gameloft Technical Game Designer Test

Project Overview
This repository contains the Unity project for the Technical Game Designer test task.

Setup Instructions
1. Unity Version: 6000.0.62f1 (Unity 6)
2. How to open:
   - Download or clone this repository.
   - Open the project via Unity Hub using the version specified above.
   - The main scene is located at: Assets/_Submission/Scenes/Tech_GD_Level_1
3. Note on Lighting:
   Large lighting data files (`.asset`) were excluded to meet GitHub's file size limits (100MB+). If the scene appears dark or has visual artifacts, please re-bake the lighting (Window > Rendering > Lighting > Generate Lighting).
